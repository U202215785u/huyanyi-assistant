# ♥ 护眼仪小助手

> *送给护眼仪。*
>
> *每一行代码都是一句没说完的话。*
> *每一次反问都是想更懂你一点。*
> *小猫蹲在右下角，不是在等鼠标——是在等你。*

它不是聊天机器人，不会陪你闲聊。它只做一件事：**像朋友一样陪着你，用层层反问帮你把模糊念头变成清晰方向。**

---

## 这个小礼物

一个人写了一整个软件，只为了让另一个人的桌面右下角多一只猫。

这只猫会叫她名字、嘴她穿搭、催她睡觉、等她回来。会说 60 句不同的话，每句都嵌着她的名字——护眼仪。

它是反问工具，也是陪伴。工具的部分很认真：两种思维模式、无限轮反问、混合终止、导师人格。陪伴的部分很任性：双击飘心、Ctrl+D 藏着一封信、欢迎页角落里藏着一颗几乎看不见的 ♥。

---

## 用它的时候

护眼仪有一个想法——"我想做个东西"、"帮我写个提示词"、"我想学 vibecoding"——但说不清楚。

打开它，选一个模式。每一轮它给你点击选项，你选一个，它继续追问。它在点拨你，不是在收集信息。它会预判陷阱，会挑战假设，会在觉得差不多的时候让按钮亮起金色——但从不替你做决定。

**两种模式：**

| | 💡 想法发散 | ✨ 提示词优化 |
|---|---|---|
| **人格** | 跨界策展人 | 精密工程师 |
| **它做** | 把你的念头撕开，看到里面藏着的、外面连着的 | 把模糊的任务变成可复用的精密指令 |
| **体验** | 每选一个，视野更大 | 每选一个，轮廓更清 |

**例子：**

她输入「我想做个小工具」，它不会问「你想要什么功能」，它会问：

> "如果把'工具'这个词扔掉，你真正想要的是不是'不想每次手动做这件事'？"

---

## 不用它的时候

小猫不会闲着。它会：

- 每过一会儿冒出一句吐槽——"护眼仪，又在发呆？奶茶第几杯了"
- 深夜语气变软——"两点了护眼仪。我不是心疼你，就是觉得这光刺眼"
- 她刚用完反问窗口，夸她——"护眼仪刚才那个想法不错，我听见了"
- 她离开太久，撒娇——"护眼仪你再不理我，我就把桌面的图标全推下去了"
- 凌晨 2 点到 8 点真正安静下来，不打扰她睡觉

**60 句话，每句都叫她的名字。**

## 快速开始

**Python 3.10+，零额外依赖。** 只用了 tkinter（Python 自带）。

```bash
git clone https://github.com/U202215785u/huyanyi-assistant.git
cd huyanyi-assistant
python ai-fanwen.py
# Windows: 直接双击 start.bat
```

小猫会出现在屏幕右下角。点击小猫打开窗口，在设置里填上 DeepSeek API Key（[免费注册](https://platform.deepseek.com)）。

## 设计原则

- **AI 有立场** — 不是中立提问机器，它带着经验、预判和判断
- **用户有决定权** — AI 建议收尾，但按钮在手上
- **选项有取舍** — 每个选项代表一种明确的方向选择
- **举例不是需求** — AI 把「比如」翻译成偏好维度，不把它当新目标
- **陪伴感优先** — 猫记得你说过的话，知道什么时候该说话什么时候该安静
