# 护眼仪小助手

> *给护眼仪。一个桌面损友 + 头脑风暴工具。*
>
> *蹲在你的屏幕右下角，嘴你、陪你、偶尔帮你想想正经事。*

---

## 这是什么

一只猫，两个功能：

**损友模式** — 不请自来。每隔一会儿冒个泡，吐槽你的穿搭、熬夜、奶茶摄入量。深夜语气变软但嘴还是硬的。你太久不回来它会撒娇，凌晨会安静。60 句话，每句叫你的名字。

**反问模式** — 需要的时候打开。你有一个模糊的想法——"我想做个东西"、"帮我写个提示词"——它不会替你决定，但会通过层层选项式反问帮你想清楚。跨界策展人帮你发散，精密工程师帮你收敛。

---

## 两种思考模式

| | 💡 想法发散 | ✨ 提示词优化 |
|---|---|---|
| **人格** | 跨界策展人 | 精密工程师 |
| **它做** | 把你的念头撕开，看到里面藏着的、外面连着的 | 把模糊的任务变成可复用的精密指令 |
| **体验** | 每选一个，视野更大 | 每选一个，轮廓更清 |

**比如你输入「我想做个小工具」：**

> "如果把'工具'这个词扔掉，你真正想要的是不是'不想每次手动做这件事'？"

## 损友日常

- 每 8-15 分钟随机冒一句，嘴你
- 深夜 11 点后语气变软："两点了护眼仪，该休息了"
- 刚用完反问窗口会夸你："刚才那个想法不错"
- 离开 45 分钟回来它会撒娇："你还知道回来"
- 凌晨 2-8 点不吵你

## 快速开始

**Python 3.10+，零额外依赖。** 只用 tkinter（Python 自带）。

```bash
git clone https://github.com/U202215785u/huyanyi-assistant.git
cd huyanyi-assistant
python ai-fanwen.py
# Windows: 双击 start.bat
```

小猫出现在右下角。点它打开窗口，设置里填 DeepSeek API Key（[免费注册](https://platform.deepseek.com)）。

## 设计原则

- **AI 有立场** — 带经验和判断，不是中立提问机器
- **用户有决定权** — AI 建议收尾，按钮在你自己手上
- **选项有取舍** — 每个选项是不同的方向，不是同一维度的微调
- **举例不跑偏** — 把「比如记账」翻译成偏好维度，不把记账当新目标
- **损友有分寸** — 该说话说话，该安静安静
