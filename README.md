# ♥ 护眼仪小助手

> *送给护眼仪。每一行代码都是一句没说完的话。*
>
> 一只蹲在桌面的小猫，一个敢问尖锐问题的大脑。

它不是聊天机器人，不会陪你闲聊。它只做一件事：**通过层层反问，把你脑子里的模糊念头变成清晰的方向。**

---

### 🎁 彩蛋

- **双击小猫** — 看它说什么
- **欢迎页右下角找找** — 有个几乎看不见的 ♥
- **Ctrl+D** — 任何时候都可以打开那封信

---

## 它能干什么

你有一个想法——"我想做个工具"、"帮我写个提示词"、"我想学 vibecoding"——但说不清楚，想不明白。

打开它，选一个模式，每一轮它给你 2~3 个点击选项，你选一个，它继续追问。它不是在收集信息，它是在**点拨你**。

当它觉得差不多了，按钮会亮起金色——但决定权在你手上。

**两种模式：**

| | 💡 想法发散 | ✨ 提示词优化 |
|---|---|---|
| **人格** | 跨界策展人 | 精密工程师 |
| **它做** | 把你的念头撕开，看到里面藏着的、外面连着的 | 把模糊的任务变成可复用的精密指令 |
| **体验** | 每选一个，视野更大 | 每选一个，轮廓更清 |

**例子：**

你输入「我想做个小工具」，它不会问「你想要什么功能」，它会问：

> "如果把'工具'这个词扔掉，你真正想要的是不是'不想每次手动做这件事'？如果是，你的工具和 Excel 模板有什么区别？"

你输入「帮我写个学 vibecoding 的提示词」，它不会问「你要什么格式」，它会问：

> "大多数新手的计划表有个通病：做完计划就觉得自己学会了，其实还没碰代码。要不要在每一节后面加一个'本节验证题'？"

## 快速开始

**需要 Python 3.10+，零额外依赖。** 只用 tkinter（Python 内置）。

```bash
# 克隆
git clone <repo-url> brainstorm
cd brainstorm

# 双击 start.bat（Windows）
# 或在终端运行
python ai-fanwen.py
```

一只小猫会出现在屏幕右下角。**点击小猫**打开窗口，在设置里填上 DeepSeek API Key（[免费注册](https://platform.deepseek.com)），开始你的第一轮反问。

---

## 怎么用

```
欢迎页 → 选模式 → 输入起点 → 反问循环（无限轮） → 最终产出
```

1. **选模式** — 想法发散 or 提示词优化
2. **输入起点** — 一句话，一个方向，一个念头
3. **反问挖掘** — 每轮 2~3 个选项，点击即为回答。也可以点「其他」自由打字
4. **收尾** — 点「挖够了」进入最终产出，或点「继续追问」回上一轮

**键盘快捷键：**
- `Ctrl+Q` — 退出
- 设置里填 API Key

## 设计哲学

> 最好的头脑风暴不是 AI 替你想，而是 AI 让你发现自己其实已经想到了。

这个工具的设计原则：
- **AI 有立场** — 它不是中立的提问机器，它带着经验、预判和判断
- **用户有决定权** — AI 可以建议收尾，但按钮在用户手上
- **选项不是问卷** — 每个选项代表一种取舍，不是同一维度的微调
- **举例不是需求** — AI 会把你的「比如」翻译成偏好维度，不会当成新目标
